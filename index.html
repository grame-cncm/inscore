About

INScore

An environment for the design of interactive augmented music scores.

INscore extends the music representation to arbitrary graphic objects:

  • Symbolic music notation [ GMN, MusicXML]
  • Textual elements
  • Bitmaps [jpg, gif, tiff, png,...]
  • Vectorial graphics (rectangles, ellipses, SVG,...)
  • Video files
  • Sound and gesture graphic representations

Main issues addressed:

  • Open graphic and time space
  • Time synchronization in the graphic space
  • Sound and gesture graphic representations
  • Interaction music processes representation
  • Dynamic and interactive scores
  • Network and web extensions

INScore is:

  • A standalone score viewer
  • An open source C/C++ library hosted on github
  • Multi-platform [MacOSX, Linux, Windows, Android, iOS]
  • An Open Sound Control API
  • An OSC based scripting language

Embedded external components:

  • GUIDO Engine - [GRAME]
  • FAUST Compiler - [GRAME]
  • Gesture Follower - [IRCAM]
  • The Qt Javascript engine - [Qt]
