This score contains an audio DSP.

It will interact with the microphone and audio of the connected clients. Some parameters of this Audio DSP will change automatically when the cursor enters SECTIONS represented by numbered markers. (1, 2, 3, 4) Clicks on the score can change the cursor of section making the activated parametter change. To trigger theses events manually, you can press theses keyboard shortcuts:

Action Key
Activate section 1 1
Activate section 2 2
Activate section 3 3
Activate section 4 4
Next section 5

This score supports Midi inputs

You can use Midi Inputs to cycle through Sections. You can define which type of Midi input will trigger the cycling. It can be :

  • Any KeyOn
  • Program change
  • Control change

INScore remote control

Clients can connect to a fildesoi.inscore native score launched in standalone. To connect to a native score, use the connection form at the end of the client page.
