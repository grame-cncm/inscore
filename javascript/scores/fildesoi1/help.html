FilDeSoi 1 - INScore

Contains an audio DSP

The Audio DSP will interact with the microphone and the audio of the connected clients. Some parameters of this Audio DSP will change automatically when the cursor enters SECTIONS represented by numbered markers. (1, 2, 3, 4) Clicks on the score can move the cursor thus changing the current section changing the active parametters. To trigger theses events manually, User can press keyboard keys that match with the section number.

Action Key
Activate section 1 1
Activate section 2 2
Activate section 3 3
Activate section 4 4
Next section →
Previous section ←

The user can use MIDI inputs, if the browser supports it.

MIDI inputs

MIDI Inputs can be used to change sections manually. User can define which type of MIDI input will trigger the change.

  • Any KeyOn
  • Program change
  • Control change

INScore remote control

Clients can connect to a fildesoi.inscore native score launched in standalone. To connect to a native score, the connection form is at the end of the client page.
