Fil De Soi 1

Fil de soi 1 and 2 are two mixed pieces for guitar and live electronics composed by Alain Bonardi and performed by Amèlia Mazarico. Fil de soi 1 was premiered on September 17 2016.
More information about the piece...

"Fil de soi 1" can be performed using the musical score available online, including all the live electronics. The musical score is interactive. It has been designed using INScore, it embeds the live electronics using the Faust library.
INScore and Faust are both projects developped by the GRAME-CNCM research laboratory.

How to use the musical score

The score provides a lot of control over the way to perform the piece. It includes a recording of the piece by Amèlia Mazarico that you can play with or without the electronics.
The piece consists of four sections, indicated on the score by the markers 1 to 4. For each section, different parameters are applied to the electronics. These parameters are automatically applied in 'play' mode when the cursor, which indicates the current position, enters a section or when you move the current position by clicking on the score. However, you can arbitrary move to any section parameters using the keyboard or using a MIDI controler (independently of the cursor position).

The figure below summarizes the various controls available to play the piece. In addition (and not part of the figure), you can also use the keyboard to control the score.

Keyboard control

You can press keyboard keys to control the score:

  • 'space bar' : start or stop the player
  • '1' to '4' : activates the parameters of the corresponding section
  • 'right arrow' : switch to next section parameters
  • 'left arrow' : switch to previous section parameters

    • MIDI control

    MIDI control is available if your browser supports the Web MIDI API and if a MIDI controler is connected to your computer. MIDI events can only be used to switch to the next section parameters. You can select the type of event between:

  • 'Program Change' : any Program Change message will trigger the next section.
  • 'Key On' : any Key On message will trigger the next section.

    • Audio routing

    It's the user responsibility to decide how the different audio streams (the recording of the piece and the audio input) are handled. The bottom right control panel is provided in this intend (where 'Playback' refers to the embedded recording of the piece):

  • 'Playback to audio output' : when ON, sends the recording to the speakers.
  • 'Playback to DSP' : when ON, sends the recording to the electronics.
  • 'Audio input to DSP' : when ON, sends the audio input (your microphone if any) to the electronics.

    • Note that you can use any arbitrary combination of selectors.

    Use cases

    Listening the recording with the electronics

  • select 'Playback to audio output'.
  • select 'Playback to DSP'.

    • Playing with automatic sections changes

  • switch 'Playback to audio output' to OFF.
  • switch 'Playback to DSP' to OFF.
  • select 'Audio input to DSP'.

    • and start the player...
    Note that you are constraigned to the tempo of the recording.

    Playing with manual sections changes

  • switch 'Playback to audio output' to OFF.
  • switch 'Playback to DSP' to OFF.
  • select 'Audio input to DSP'.

    • don't start the player but play your instrument...
    Note that your tempo is free, but you have to switch manually to the next section using the keyboard or a MIDI controler.
