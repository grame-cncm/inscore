FilDeSoi

Fil de soi is a mixed piece for acoustic and electronic guitar in real time. Composed by Alain Bonardi and performed by Am√®lia Mazarico. Created the 17th september 2016. Inscore allow this score to embed the audio DSP of the composition and connect the different inputs to this DSP.

Audio connections configuration

In The bottom right corner, the user can configure which input is connected to what output. The Three modes can be activated together:

  • Wav audio to audio Output will connect the wav audio to the Output.
  • Wav audio to DSP will connect the wav audio to the DSP.
  • Audio In to DSP will record the microphone of the user (asked permission at the start of the page) and inject it to the DSP.
In any case the DSP is connected to audio Output.

InscoreWeb

The INScoreJS library allows to embed the INScore engine services in a web page and to design augmented, dynamic and interactive musical scores that can be published on the Internet.

Faust

Faust (Functional Audio Stream) is a functional programming language for sound synthesis and audio processing with a strong focus on the design of synthesizers, musical instruments, audio effects, etc. Faust targets high-performance signal processing applications and audio plug-ins for a variety of platforms and standards. Thanks to INScore, we can embeded a Faust DSP allowing real time signal processing inside the augmented score.

Contains an audio DSP

The Audio DSP will interact with the microphone and the audio of the connected clients. Some parameters of this Audio DSP will change automatically when the cursor enters SECTIONS represented by numbered markers. (1, 2, 3, 4) Clicks on the score can move the cursor thus changing the current section changing the active parametters. To trigger theses events manually, User can press keyboard keys that match with the section number.

Action Key
Activate section 1 1
Activate section 2 2

Activate section 4 4
Next section →
Previous section ←

The user can use MIDI inputs, if the browser supports it.

MIDI inputs

MIDI Inputs can be used to change sections manually. User can define which type of MIDI input will trigger the change.

  • Any KeyOn
  • Program change
  • Control change

INScore remote control

Clients can connect to a fildesoi.inscore native score launched in standalone. To connect to a native score, the connection form is at the end of the client page.
