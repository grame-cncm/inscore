Manual de Referência de Lua 5.1

Essa é uma tradução para o português do manual de referência de Lua, que é a definição oficial da linguagem Lua. Para uma introdução completa à programação em Lua, veja o livro Programming in Lua.

A versão original em inglês também está disponível como livro:

Lua 5.1 Reference Manual
by R. Ierusalimschy, L. H. de Figueiredo, W. Celes
Lua.org, August 2006
ISBN 85-903798-3-3

Comprando esse livro você ajuda a apoiar o projeto Lua.

A tradução para o português foi feita por Sérgio Queiroz de Medeiros com revisão de Roberto Ierusalimschy.
